# Exemple : redaction d'une note de version

> Statut : stable | Derniere maj : 2026-06-24 | Auteur : Cerveau Partage | Mots-cles : prompt, note de version, procedure, exemple

Fiche d'exemple. Elle montre le format attendu dans `04_skills/`. Remplacez son contenu ou supprimez-la.

## Quand l'utiliser
Pour transformer une liste de tickets resolus en note de version lisible par les clients.

## Procedure
1. Regrouper les changements en trois categories : nouveautes, ameliorations, corrections.
2. Une phrase par element, orientee benefice utilisateur, sans jargon technique.
3. Mettre en avant le changement le plus visible en premier.

## Exemple de formulation
Plutot que "fix bug cache", ecrire "Les modifications recentes apparaissent maintenant immediatement".
